# JS-To-Do
A simple, responsive, and minimal to-do list web app built using HTML, CSS, and JavaScript.

## Features

- Add new task
- Mark tasks as completed
- Delete Tasks
- Duplicate Tasks with unique ID 
- Simple and minimal user interface
- Responsive Design, supports mobile view
- Stores the task using local storage

## Tech Stack

- HTML
- CSS
- Tailwind CSS
- JavaScript

## Known Limitations
  
- No edit option for existing tasks
- No due dates or priority features
- No filter options for completed or pending tasks
- Only one layout is currently available
